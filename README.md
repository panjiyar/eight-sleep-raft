# eight-sleep-raft

## Project Description

## Prerequisites

- Python 3

## Setup

```bash
# Clone the repo
git clone <repo-url>
cd eight-sleep-raft

# Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Running Tests

```bash
pytest test_raft.py
```
